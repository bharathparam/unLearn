# ⬡ NeuralLifecycle Framework

**Ultra-modern 3D Cyberpunk AI Research Dashboard**

> *"A live operating system for controlling and editing AI memories."*

---

## 🚀 Quick Start

```bash
# Install dependencies
pip install streamlit plotly altair pandas numpy scipy scikit-learn reportlab pillow

# Run the app
python -m streamlit run app.py
# or double-click run.bat
```

Open **http://localhost:8501** in your browser.
